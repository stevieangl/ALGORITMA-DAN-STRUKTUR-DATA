binary searching program
Dengan program searching binary,  program akan mencari angka yang diinput lalu mencari nilai tengah array, kemudian mencari angka yang cocok dengan mempersempit kemungkinan lokasi.
Jika angka yang dicari ada, maka program akan memberitahukan lokasi/index dari angka tersebut dalam array. Jika angka yang dicari tidak ada, maka program akan menampilkan "Angka tersebut tidak ditemukan".


searching program
program searching ini akan mencari dan menampilkan posisi dari data yang dicari dalam array. Penambahan perintah "jumlah++" digunakan untuk mendeteksi data yang dicari dalam data berjumlah lebih dari satu. Jika data yang dicari tidak ada, maka program akan menyatakan bahwa data tidak ditemukan. Setelah itu, program akan menampilkan jumlah data yang ditemukan dalam array.
